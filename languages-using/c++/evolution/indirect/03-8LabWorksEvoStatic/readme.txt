Эволюционное расширение с использованием моделирования процедурно-параметрического стиля со статическим заданием признаков специализаций
------------------------------------------------------------------------------------------------------------------------------------------------

В данном каталоге делается попытка реализовать все восемь лабораторных работ с использованием эволюционного расширения базовой работы по принятой в методичке схеме. То есть, когда все расширения идут в основном от первой работы, за исключением тех работ, которые требуют предварительной реализации ранее созданных конструкций.
Общая идея заключается в том, чтобы первая работа учитывала все возможности расширения, проводимые в последующих работах.
При этом изначально делается попытка ориентации на процедурно-параметрический подход.
Все работы пишутся с нуля относительно новых вариантов обобщений.

Условие задачи
--------------------

Даны геометрические фигуры.
Необходимо прочитать их из файла в виде удобном для компьютерной обработки и разместить в контейнере на основе одномерного массива.
После этого содержимое контейнера вывести в другой файл в виде, удобном для восприятия человеком.

Изначально даны следующие фигуры:

    1) Треугольник со сторонами a, b, c.
    2) Прямоугольник со сторонами x, y.

Контейнер на основе одномерного массива заданной размерности превышение которой приводит к прекращению ввода и вывода ограниченного числа введенных фигур.
